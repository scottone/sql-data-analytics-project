# sql-data-analytics-project
This repository contains a collection of SQL scripts demonstrating various analytical techniques, such as changes over time, cumulative, performance, data segmentation, part-to-whole analysis.

These documents were collected while completing the SQL Data Analytics project portion of this Udemy course (Jan 2026 version):
The Complete SQL Bootcamp (30 Hours) - Go from Zero to Hero

Thanks to the instructor, <https://qttechacademy.udemy.com/user/baraa-khatib-salkini-2/>, for the excellent, engaging instruction and challenging projects.


The datasets used are the gold dimension and fact tables from the sql-data-warehouse-project.
